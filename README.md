# Python Scientific Calculator

## Description

This is a menu-driven calculator application developed using Python. It performs basic arithmetic operations as well as advanced mathematical functions. The calculator provides a simple command-line interface and includes error handling for invalid mathematical operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Exponentiation
* Floor Division
* Square Root
* Cube Root
* Logarithm
* Sine Function
* Cosine Function
* Tangent Function
* Cotangent Function
* Error Handling for Invalid Operations
* Continuous Calculation Mode
* User-Friendly Menu Interface

## Technologies Used

* Python 3
* Math Module

## How to Run

1. Clone or download the project.
2. Open a terminal in the project directory.
3. Run the program:

```bash
python calculator.py
```

4. Select an operation from the menu.
5. Enter the required values.
6. View the result.
7. Choose Exit to close the application.
